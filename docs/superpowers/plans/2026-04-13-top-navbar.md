> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** Introduce a reusable `TopNav` component rendered by each major route. Keep existing route-specific links in place, remove only the standalone homepage logo block, and share navbar styling through the existing module CSS structure.

**Tech Stack:** React, React Router, CSS Modules, Testing Library, Vitest, Vite
