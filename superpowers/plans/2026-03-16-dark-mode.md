# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert the site to an always-dark theme using CSS custom properties.

**Architecture:** Define all theme colors as CSS custom properties in `colors.css`, apply the background and text color to `body` in `globals.css`, and replace two hardcoded color values in `App.module.css`.

**Tech Stack:** CSS custom properties, Vite

**Spec:** `superpowers/specs/2026-03-16-dark-mode-design.md`

---

**Files:**

### Task 2: Apply theme to globals.css

**Files:**
- Modify: `src/styles/globals.css`

The current `html, body { ... }` rule sets font and padding but has no background or text color. Add both.

- [ ] **Step 1: Add background and text color to the `html, body` rule**

The rule should become:

```css
html,
body {
  padding: 0;
  margin: 0;
  font-family: -apple-system, BlinkMacSystemFont, Segoe UI, Roboto, Oxygen,
    Ubuntu, Cantarell, Fira Sans, Droid Sans, Helvetica Neue, sans-serif;
  scroll-behavior: smooth;
  background-color: var(--bg);
  color: var(--text-primary);
}
```

- [ ] **Step 2: Commit**

```bash
git add src/styles/globals.css
git commit -m "style: apply dark background and text color to body"
```

### Task 3: Fix hardcoded colors in App.module.css

**Files:**
- Modify: `src/App.module.css`

Two hardcoded color values need replacing:
- `.container a { color: black }` → `color: var(--link-color)`
- `.container hr` border `rgb(190, 190, 190)` → `var(--hr-color)`

- [ ] **Step 1: Replace the file**

```css
.container {
  padding: 1rem 1rem;
  display: flex;
  flex-direction: column;
  align-items: center;
}

.container main {
  max-width: 1200px;
}

/* Styling for all hr elements */
.container hr {
  border-top: 1px dashed var(--hr-color);
  border-bottom: none;
  border-left: none;
  border-right: none;
  width: 60%;
}

/* Styling for all a elements (clickable links) */
.container a {
  text-decoration: none;
  border-bottom: 2px dotted var(--blue-emphasis);
  color: var(--link-color);
}
.container a:hover {
  color: var(--blue-emphasis);
  opacity: 0.8;
}
```

- [ ] **Step 2: Commit**

```bash
git add src/App.module.css
git commit -m "style: replace hardcoded colors with CSS custom properties"
```
